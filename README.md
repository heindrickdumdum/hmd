# Heindrick Dumdum — Portfolio

Personal portfolio website built for GitHub Pages.

## Deploy to GitHub Pages

1. Create a new GitHub repo (e.g. `heindrickdumdum.github.io` for root, or any repo name)
2. Upload `index.html` to the repo
3. Go to **Settings → Pages**
4. Set Source to **Deploy from a branch → main → / (root)**
5. Save — your site will be live at `https://yourusername.github.io/reponame`

## Files
- `index.html` — The entire portfolio (self-contained, no build step needed)

## Customization Tips
- Update the contact form email in the `handleForm()` function
- Add real project URLs/links in the Projects section
- Swap the profile photo by replacing the base64 image data
